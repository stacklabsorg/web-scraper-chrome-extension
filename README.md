A **TypeScript-based Chrome Extension** for user-assisted DOM scraping.  
Select elements on any website and instantly extract structured data for export or further processing.

---

## Features

- **User-assisted scraping**: Click on any element to capture its text and metadata.
- **DOM element highlighting**: Hovered elements are highlighted for easy selection.
- **Data storage**: Collected data is stored in Chrome storage for easy retrieval.
- **Clear & export**: Clear scraped data or prepare it for export (CSV/JSON integration ready).
- **Lightweight & fast**: Written in TypeScript, optimized for Chrome MV3.
- **Reusable**: Works on any website with a generic DOM selection method.
